# Support

If you need help with terraform-provider-flashduty, here are some resources:

## Getting Help

- **Documentation**: Check the [Terraform Registry documentation](https://registry.terraform.io/providers/flashcatcloud/flashduty/latest/docs)
- **Bug Reports**: Open an issue on [GitHub Issues](https://github.com/flashcatcloud/terraform-provider-flashduty/issues)
- **Feature Requests**: Open an issue on [GitHub Issues](https://github.com/flashcatcloud/terraform-provider-flashduty/issues)
- **Security Issues**: Please report security vulnerabilities via email to support@flashcat.cloud (see [SECURITY.md](SECURITY.md))

## Community

- [Flashcat Cloud](https://flashcat.cloud)
